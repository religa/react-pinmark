# Changelog

## [0.1.0] — 2026-02-27

Initial release.

### Features

- Pin-based commenting overlay for React applications
- Threaded replies with author names and relative timestamps
- Markdown rendering in comment bodies
- Image attachments (file upload and clipboard paste)
- Auto viewport screenshot on pin placement
- Slide-in thread list panel with page/status filtering
- Dark mode (`light` / `dark` / `system`)
- Supabase adapter (`react-pinmark/supabase`)
- Custom backend adapter interface
- `useComments()` hook for full programmatic control
